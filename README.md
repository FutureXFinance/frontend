# FutureX Finance - Frontend

AI-powered financial services platform - Frontend application built with Next.js 14.

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Axios for API calls

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## Environment Variables

Create `.env.local`:

```bash
NEXT_PUBLIC_API_URL=http://localhost:8000
NEXT_PUBLIC_API_V1=/api/v1
```

## Features
- KYC Verification Interface
- Fraud Detection Dashboard
- Document Parser
- AI Chatbot
- Customer Insights
- Compliance Monitoring

## Deployment

Deploy to Vercel:
```bash
vercel --prod
```
